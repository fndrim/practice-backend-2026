# Survey API — Система управления опросами

Бэкенд-приложение на Laravel для создания опросов, сбора ответов и аналитики результатов. Реализовано в рамках учебной практики (Чекпоинты 1-4).

## 🚀 Функциональные возможности:

1.  **Проектирование БД**: Реализована структура таблиц с форейн-ключами (Surveys -> Questions -> Answers).
2.  **API для опросов**:
    * `GET /api/surveys` — Получение списка всех доступных опросов.
    * `GET /api/surveys/{id}` — Получение детальной информации об опросе вместе с его вопросами.
3.  **Прием ответов**:
    * `POST /api/surveys/{id}/answers` — Сохранение ответов пользователей в базу данных.
4.  **Аналитика (Advanced)**:
    * `GET /api/surveys/{id}/results` — Сводная статистика по опросу (общее кол-во участников и список ответов по каждому вопросу).

## 🛠 Технологии
* **PHP 8.x**
* **Laravel 11/12**
* **MySQL** (XAMPP)
